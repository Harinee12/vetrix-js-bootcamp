Git Workflow Documentation
Repository Creation

I created a GitHub repository named "vetrix-js-bootcamp"to store all my JavaScript Bootcamp tasks and practice work. This helps me keep the project organized and maintain version control.

Creating My Branch

I created my personal branch named "harini"to work independently without affecting the main branch.

Command used:

git checkout -b harini
 Pushing Code

After completing the task, I added the files, committed the changes, and pushed them to GitHub.

Commands used:

git add .

git commit -m "Added Day 2 JavaScript Bootcamp Task"

git push -u origin harini

Adding Collaborators

The repository is shared among three members. Collaborators can be added using their GitHub usernames, and each member can create their own branch to push daily work.

Working with Multiple Branches

Different branches can be used for different tasks or projects.

This keeps the work organized and allows multiple people to collaborate easily.
